# Formula 1 Winner Prediction using Machine Learning

## Project Overview

This project uses historical Formula 1 race data to predict race winners using Machine Learning models.

The project demonstrates the complete ML workflow from data collection to model optimization and prediction.

---

## Dataset

The dataset contains:

- Driver information
- Constructor information
- Race results
- Driver points
- Grid positions

Data was collected from Formula 1 historical race records.

---

## Project Workflow

### 1. Data Collection
Collected and prepared Formula 1 race data.

### 2. Feature Engineering
Created features such as:

- Grid position
- Driver points
- Average driver points

### 3. Logistic Regression Model
Built the first baseline prediction model.

### 4. Model Evaluation
Evaluated model performance using:

- Accuracy Score
- Confusion Matrix
- Classification Report

### 5. Model Improvement
Added new features to improve predictions.

### 6. Model Comparison
Compared multiple machine learning algorithms.

### 7. Random Forest Model
Implemented Random Forest classification.

### 8. Feature Importance Analysis
Identified the most influential features.

### 9. Winner Prediction
Predicted future race winners using model probabilities.

### 10. Hyperparameter Tuning
Optimized Random Forest using GridSearchCV.

### 11. Final Project Summary
Compared all models and summarized results.

---

## Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- Tuned Random Forest Classifier

---

## Results

| Model | Accuracy |
|---------|----------|
| Logistic Regression | 96.06% |
| Decision Tree | 99.46% |
| Random Forest | 99.44% |
| Tuned Random Forest | 99.57% |

Best model: **Tuned Random Forest**
